# RelayForge Brand Guide (v1)

## Primary Identity
- Primary logo direction: **RF monogram + RelayForge wordmark** (Prompt 2 direction)
- Secondary motif: abstract forged-circuit motif (Prompt 3 style) for marketing visuals only

## Tagline
**Forge reliable browser automation.**

## Color Tokens
- RF Blue: `#2563EB`
- RF Navy: `#0B1220`
- RF White: `#FFFFFF`
- RF Slate: `#94A3B8`
- RF Amber (accent): `#F59E0B`

## Typography
- Primary: Inter
- Fallback: system-ui, Arial, sans-serif

## Logo Rules
- Clear space: at least 1x the R-stem height around mark
- Minimum sizes:
  - Horizontal lockup: 120px width
  - Icon only: 20px
- Required variants:
  1. Full color (light)
  2. Full color (dark)
  3. Monochrome black
  4. Monochrome white
  5. Icon-only full color
  6. Icon-only monochrome

## Do / Don't
### Do
- Keep mark simple and high contrast
- Use approved color palette
- Keep icon and wordmark spacing consistent

### Don't
- Stretch or warp logo
- Add heavy glows/shadows
- Replace primary logo with motif artwork
