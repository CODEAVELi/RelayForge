# Beta Signup Template

## Intake fields
- Name
- Email
- Primary browser (Chrome/Firefox/Safari)
- Primary use case
- Avg actions/day
- Biggest current relay pain point
- Consent to share anonymized diagnostics (Y/N)

## Acceptance criteria (first cohort)
- Uses browser automation at least 3x/week
- Will provide at least one bug report or feedback note
- Comfortable with beta software

## Onboarding message
Thanks for joining Relay Next beta.

Install docs: INSTALL.md
Quickstart video script: QUICKSTART_VIDEO_SCRIPT.md

Please run one 15-minute workflow and send:
1) What failed (if anything)
2) Diagnostics JSON export
3) Browser + OS version

We ship fixes weekly and will keep you posted in release notes.
